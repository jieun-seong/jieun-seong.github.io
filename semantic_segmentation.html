Semantic Segmentation 
Using Deep Learning

Implement the PSPNet architecture introduced in the paper, Pyramid Scene Parsing Network. 
This architecture uses a ResNet backbone, but uses dilation to increase the receptive field, and aggregates context over different portions of the image with a "Pyramid Pooling Module" (PPM). 

Pyramid Pooling Module

Source: https://arxiv.org/pdf/1612.01105.pdf

Dilated Convolution

Source: https://github.com/vdumoulin/conv_arithmetic#dilated-convolution-animations 

Dilated convolution has larger receptive fields at the same computational cost as the traditional convolution. Also, there is no loss of resolution in the output images.
